#### [Unsupervised Anomaly Detection for Time Series](https://github.com/anna1kharchenko/unsupervised-time-series-anomaly-detection)
*Investigation into the effectiveness of various architectures for detecting anomalies in server metrics (System Load, Memory Usage, etc.).*

* **Technical Approach:** Implemented a synthetic anomaly generation pipeline based on the **CAPMix (Context-Aware Pattern Mixup)** methodology to enable robustness testing against complex contextual and point anomalies.
* **Models Implemented:** Built and benchmarked **CNN-LSTM Autoencoders**, **Isolation Forest**, and **One-Class SVM**.
* **Analytics & Evaluation:** Utilized MSE, ROC-AUC, and Precision-Recall metrics to evaluate model performance in reconstructing normal states and identifying deviations.

#### [Medical Data Clustering Analysis](https://github.com/anna1kharchenko/clustering-analysis-medical-data)
*Unsupervised patient profiling through multi-model clustering.*

* **Methodology:** Benchmarked **K-Means**, **DBSCAN**, **Affinity Propagation**, and **Feature Agglomeration**.
* **Dimensionality Reduction:** Utilized **PCA** for feature extraction and **t-SNE** for high-dimensional medical data visualization.
* **Evaluation:** Validated cluster stability and consistency using **ARI** (Adjusted Rand Index) and **AMI** (Adjusted Mutual Information) metrics.

#### [Gallstone Correlation Analysis & SVM Classification](https://github.com/anna1kharchenko/Gallstone-Correlation-Analysis-SVM)
*Predictive modeling and feature correlation study using UCI medical datasets.*

* **Analysis:** Constructed correlation matrices and performed feature importance/weight analysis.
* **Modeling:** Implemented **Support Vector Machine (SVM)** with hyperparameter optimization for precise clinical state classification.
